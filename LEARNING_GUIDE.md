## 🎯 課程簡介

本課程為 **6週地震學與地球物理主題課程**，結合 **AI/Agent 輔助教學** 的實作專案。目標是將傳統教學延伸為動手實作，並將地震學和地球物理知識轉化為中小學課堂中可立即應用的互動教材。

**課程特色**：
- ✅ 理論 + 實作並重（40% 理論，60% 實作）
- ✅ 結合 AI 工具輔助教學
- ✅ 強調動手實作與自動化流程
- ✅ 產出可實際應用的互動教學資源

---

## 📌 核心學習原則

### 1. 動手實作 > 理論背誦
- 這門課 **60% 是實作**，40% 是理論
- 每週都有 **具體產出目標**
- **建議**：每週預留 **4-6小時** 專門用於實作練習

### 2. 工具熟練度是關鍵
- **Week 1 的工具基礎** 是後續所有週次的前提
- 如果 Git、Python、VS Code 不熟，後面會非常吃力
- **建議**：在 Week 1 多花 2-3 倍時間，確保工具使用流暢

## 🎯 通用學習建議

### 1. 建立學習社群 👥
- 找 **2-3 位同伴** 組成學習小組
- 定期 **分享進度、互相幫助**
- 可以建立 **Discord 或 Line 群組** 討論

### 2. 利用 AI 工具輔助 🤖
- 使用 **GitHub Copilot** 或 **ChatGPT** 幫助寫程式
- 讓 AI 解釋 **複雜的地震學概念**
- **但不要完全依賴 AI**，要自己動手實作

### 3. 保持好奇心 ❓
- 多問「為什麼？」
- 例如：為什麼 P 波比 S 波快？為什麼需要三個測站定位？
- 這些問題的答案往往就是考試重點

### 4. 及時複習 📝
- 每週結束後，花 **1-2 小時** 回顧該週內容
- 記錄 **學到的新知識** 和 **遇到的問題**
- 如果有不懂的地方， **立即請教老師或同學**

### 5. 實際應用 🌍
- 尋找 **生活中的地震相關新聞**
- 試著用所學知識 **分析新聞中的地震事件**
- 這樣可以 **加深理解** 並 **提高興趣**

## 📞 遇到問題怎麼辦？

### 技術問題
1. **Google 搜索**：大部分問題都能在 Stack Overflow 找到答案
2. **官方文件**：先查看工具的官方文件
3. **同學討論**：在學習群組中提問
4. **老師諮詢**：如果問題持續無法解決，請教老師

### 概念問題
1. **重新閱讀教材**：很多時候答案就在課本中
2. **畫圖理解**：地震學中很多概念需要視覺化
3. **舉例說明**：用具體例子幫助理解抽象概念
4. **教別人**：試著把概念講給別人聽，能幫助自己理清思路
