# 🌍 2026 北市大地球科學課程計畫

> **地震學、地球物理與 AI/Agent 輔助教學實作**
> 🏫 台北市立大學地球科學課程

---

## 📋 專案簡介

本專案是 **2026 年北市大地球科學課程計畫** 的官方倉庫，旨在將 **地震學** 與 **地球物理** 知識轉化為 **中小學課堂中可立即應用的互動教材**，並結合 **AI/Agent 輔助教學** 的創新模式。

### 🎯 核心目標

- ✅ **理論實踐化**：將複雜的地球科學理論轉化為易於理解的教學內容
- ✅ **工具自動化**：運用 AI 工具和自動化流程提升教學效率
- ✅ **互動教學**：開發可互動的教學模組和網頁應用
- ✅ **實務應用**：所有教材均可直接應用於中小學課堂

---

## 📅 課程架構

本課程為期 **6週**，每週聚焦不同主題，包含理論學習與實作練習。

| 週次 | 日期 | 主題 | 重點內容 |
|------|------|------|----------|
| Week 1 | 7/17 | 環境建立與工作流程自動化 | 建立雲端環境、學習基礎工具 |
| Week 2 | 7/24 | 地震成因、震動與加速度量測 | 板塊運動理論、感測器應用 |
| Week 3 | 7/31 | 震源機制與震央定位實作 | P/S 波分析、三角定位法 |
| Week 4 | 8/7 | 地震波形分析與導引 | 真實波形分析、教學報告製作 |
| Week 5 | 8/14 | 防災互動網頁與場址效應 | 地震危害模型、互動網頁開發 |
| Week 6 | 8/21 | 成果整合與野外教學實測 | 野外測量、教學作品包完成 |

## 🚀 快速開始

### 前置要求
- ✅ Git 客戶端
- ✅ VS Code 或其他代碼編輯器
- ✅ Python 3.8+ 
- ✅ 現代瀏覽器（Chrome、Firefox、Edge）

### 安裝步驟

1. **克隆倉庫**
   ```bash
   git clone https://github.com/mengching1213/earth_science_course_2026_mengching.git
   cd earth_science_course_2026_mengching
   ```

2. **開啟首頁**
   ```bash
   # 直接在瀏覽器中開啟 index.html
   open index.html
   ```

3. **閱讀學習指南**
   - 詳細的學習策略請參見 [LEARNING_GUIDE.md](./LEARNING_GUIDE.md)
